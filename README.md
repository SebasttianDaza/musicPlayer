# Music Player 🎶🎶🎶🎶

Web music player with different functionalities, developed with HTML, CSS, JS and React, using Desing Patterns [🖥️🖥️🏴]. 

Changing in the player:
- Playlist
- Image🏴
- Controls
- Lyrics with API 🎶🎶
- Interaction with API
- Search with API 💯💯
